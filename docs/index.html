üìö Available Endpoints

GET /api/tvshows

Description: Retrieve all TV shows with pagination support

Query Parameters:

  • page (optional, default: 1) - Page number
  • limit (optional, default: 20, max: 100) - Items per page

Example: /api/tvshows?page=1&limit=20

GET /api/tvshows/filter/year

Description: Filter TV shows by year range (first air date)

Query Parameters:

  • start_year (required) - Starting year (inclusive)
  • end_year (required) - Ending year (inclusive)
  • page (optional) - Page number
  • limit (optional) - Items per page

Example: /api/tvshows/filter/year?start_year=2008&end_year=2013

GET /api/tvshows/random

Description: Get 10 randomly selected TV shows

Query Parameters: None

Example: /api/tvshows/random
